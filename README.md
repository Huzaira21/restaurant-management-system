# Restaurant Management System

A console-based restaurant management system built in C++.
Developed during 2nd semester at Namal University, Mianwali.

## Features
- Order management
- Billing system
- Report generation
- ## Languages Used
- C++ — Main system logic
- Assembly Language — Low-level implementation

## How to Run
> **Note:** Uses Windows-specific libraries (`windows.h`, `conio.h`)
> Open in Dev C++ on Windows and press F11 to compile and run.

## Skills Used
C++ · Object-Oriented Programming · File Handling · Problem Solving
